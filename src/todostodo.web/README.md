# todostodo.web

React frontend for TodosToDo. Setup, architecture, and run instructions are in the [repository README](../../README.md).

Frontend-only commands (from this directory):

```bash
pnpm install
pnpm dev          # http://localhost:5173 (API must be on :5162)
pnpm test:ci
pnpm generate-api # API must be running; see openapi-ts.config.ts
```
